# Fair Play Claremont

**fairplayclaremont.org** — A one-page advocacy site highlighting gender inequity in Claremont youth sports and calling for the repeal of Ordinance 11.08.

Built with [Astro](https://astro.build) + [Tailwind CSS](https://tailwindcss.com). Deployed to [Cloudflare Pages](https://pages.cloudflare.com) via GitHub Actions.

---

## Tech stack

| Layer | Tool |
|---|---|
| Framework | Astro (static output) |
| Styling | Tailwind CSS v4 |
| OG image | Sharp (Node.js, runs at build time) |
| Sitemap | @astrojs/sitemap (auto-generated) |
| Hosting | Cloudflare Pages |
| CI/CD | GitHub Actions |

---

## Local development

```bash
# Install dependencies
npm install

# Start dev server (http://localhost:4321)
npm run dev
```

The OG image is generated automatically before each build. To regenerate it manually:

```bash
npm run generate-og
```

## Deploying to Cloudflare Pages

### 1. Create a Cloudflare Pages project

1. Log in to the [Cloudflare dashboard](https://dash.cloudflare.com)
2. Go to **Workers & Pages** → **Create** → **Pages**
3. Choose **"Connect to Git"** and authorize your GitHub account
4. Select the `fairplayclaremont.org` repository
5. Set these build settings:
   - **Framework preset:** Astro
   - **Build command:** `npm run build`
   - **Build output directory:** `dist`
6. Click **Save and Deploy**

### 2. Get your Cloudflare API token

1. In the Cloudflare dashboard, go to **My Profile** → **API Tokens**
2. Click **Create Token**
3. Use the **"Edit Cloudflare Workers"** template, or create a custom token with:
   - **Account** → Cloudflare Pages: Edit
4. Copy the token — you will not see it again

### 3. Get your Cloudflare Account ID

1. In the Cloudflare dashboard, go to **Workers & Pages**
2. Your **Account ID** is in the right sidebar

### 4. Add secrets to GitHub

1. Go to your GitHub repo → **Settings** → **Secrets and variables** → **Actions**
2. Add two repository secrets:
   - `CF_API_TOKEN` — the API token from step 2
   - `CF_ACCOUNT_ID` — the Account ID from step 3

### 5. Connect your custom domain

1. In Cloudflare Pages, open your project → **Custom domains**
2. Click **Set up a custom domain**
3. Enter `fairplayclaremont.org`
4. Cloudflare will guide you through the DNS record — if your domain DNS is already on Cloudflare it's automatic

After this, every push to `main` triggers a full build and deploy automatically.

## Making content updates

All field data and copy lives directly in the component files. Edit and push to `main`.

| What to update | File |
|---|---|
| Hero headline / subtext | `src/components/Hero.astro` |
| Stat bar numbers | `src/components/StatBar.astro` |
| Field data (lights, fence, lock) | `src/components/FieldComparison.astro` |
| Legal framing + asks | `src/components/LawAndAsks.astro` |
| CTA text + footer links | `src/components/CtaFooter.astro` |
| OG image design | `scripts/generate-og.mjs` then `npm run generate-og` |
| Site title / description / meta | `src/layouts/Base.astro` |
